# Universal Research Assistant

A lightweight ChatGPT App designed to help global users understand any complex topic by:

- Breaking information into simple structures  
- Explaining like a teacher  
- Providing guided reasoning paths  
- Producing summaries and diagrams  
- Helping users learn faster and more clearly  

## Features
- Concept breakdown  
- Compare-and-contrast reasoning  
- Use-case based explanation  
- Structural summaries  
- Multi-language support (input any language, output English by default)

## Why this App
Most people struggle with:
- Overwhelming information  
- Difficulty understanding technical or abstract concepts  
- Slow learning speeds  

This app solves the *cognitive bottleneck* by giving immediate clarity.

## Example Prompts (for global users)
- “Explain Ethereum to me like I'm 12.”  
- “Compare Transformers vs RNNs.”  
- “Summarize Ray Dalio’s debt cycle into 3 levels.”  
- “Why is inflation happening in Japan?”  

## Technical Notes
- Fully language-agnostic input  
- Structured English output  
- Lightweight and AI-callable
